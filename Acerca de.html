La importancia del uso de HTML y CSS para crear paginas web

Enseñando y aprendiendo

Lo que he aprendido en el tema de hacer codigos con HTML y CSS, desde Visual Studios Code (VSC), es sobre como crear paginas web desde esa aplicacion, sobre como crear un menu desde la pagina web, y que lleva hacia otra, ya sea enlace interno, de una propia pagina que haya creado, o tambien enlace externo que lleve hacia otra pagina. Tambien he aprendido sobre como hacer header, que lleve un boton que lleve hacia otra pagina, tambien a hacer un footer, que lleve nuestro nombre y los derechos de autor.

Otra cosa que aprendi durante este tiempo que llevamos viendo todo esto, ha sido sobre como usar CSS, para modificar mejor la pagina web, cambiarle el color de fondo de la pagina, modificar el tamano de letra, el color, el tipo de letra, y todas esas cosas, desde CSS, para luego ponerlo todo en una carpeta, y quede en la pagina web.
